Automating AWS Operations with Lambda

To optimize resource usage and reduce costs, I designed a serverless solution using AWS Lambda. Recognizing the need for EC2 instances to run only during specific periods, such as company working hours (8:00 AM to 5:00 PM), I implemented two Lambda functions to automate instance management. These functions, triggered by CloudWatch Events, efficiently start and stop instances at the appropriate times. This project highlights my ability to leverage AWS serverless technologies to create cost-effective and automated solutions tailored to operational needs.

Developing a Packet Sniffer in Python

To enhance my networking and programming skills, I built a packet sniffer using Python's socket module. By configuring the socket to capture packets directly from the network, I created scripts to extract and analyze the captured data. This project not only sharpened my understanding of network protocols but also provided valuable hands-on experience in cybersecurity concepts. It showcases my ability to combine programming and networking knowledge to tackle complex technical challenges.
